# Strategy — Frameworks por formato

Escolha o formato pelo objetivo + plataforma. Cada um define estrutura, duração, gancho,
edição, som e CTA. A skill mapeia `output.platform` + `intent.goal` para um destes.

## 1. Reels / TikTok / Shorts (UGC & tráfego pago)
- **Estrutura:** gancho visual → problema relacionável → solução rápida → prova social → CTA direto.
- **Duração:** 15–45s.
- **Gancho:** texto na tela + ação imediata.
- **Edição:** jump cuts rápidos, legendas animadas, **sem silêncios**.
- **Som:** áudio original claro + música de fundo / trend.
- **CTA:** link na bio ou pedir comentário específico.
- **Erro a evitar:** introdução longa.
- → preset sugerido: `viral` / `acao` · aspect 9:16 · `framework: Hook-Value-CTA`.

## 2. Venda Direta / Landing (VSL curta)
- **Estrutura:** gancho de identificação → agitar a dor → produto como herói (benefícios) → prova/garantia → CTA forte.
- **Duração:** 2–5 min.
- **Gancho:** promessa de transformação clara OU choque de realidade.
- **Edição:** profissional, B-rolls ilustrando a narrativa, ritmo médio.
- **Som:** locução confiante + música que cresce com a história.
- **CTA:** escassez / oferta especial.
- **Erro a evitar:** focar em características técnicas em vez de benefícios emocionais.
- → preset: `vendas` · aspect 16:9 ou 9:16 · `framework: Save the Cat` (ou PAS).

## 3. Autoridade / Curso online
- **Estrutura:** gancho curioso → história pessoal (Save the Cat) → grande descoberta/metodologia → valor prático → convite.
- **Duração:** 1–3 min (social) ou até 10 min (YouTube).
- **Gancho:** revelação ou quebra de mito do mercado.
- **Edição:** "talking head"/podcast de qualidade + gráficos e textos explicativos.
- **Som:** áudio de estúdio + trilha sutil que ajuda o foco.
- **CTA:** aula gratuita / inscrição no canal.
- **Erro a evitar:** aula chata sem gancho nem ritmo.
- → preset: `suave` / `promo` · aspect 16:9 · `framework: Save the Cat` + `emotion: facts`.

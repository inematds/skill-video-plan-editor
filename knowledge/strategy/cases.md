# Strategy — Biblioteca de cases (lições replicáveis)

Cases clássicos de alta performance e a lição que se transfere.

> **Caveat honesto:** quase todos são **live-action**. A lição de *gancho, narrativa e
> retenção* transfere 100%; a *execução* a gente adapta para motion graphics + b-roll
> gerado (flux2-klein) + recipes cinematográficas. Não copiamos o vídeo — roubamos o
> princípio.

| Case | Tipo / Foco | Lição replicável |
|---|---|---|
| **Blendtec — Will It Blend?** | Demonstração extrema / viral | **Demonstração extrema do produto**: se é potente, mostre de um jeito que pareça quase impossível. Curiosidade pura segura ("será que mistura?"). |
| **Dollar Shave Club** | Humor absurdo / venda direta | **Humor autodepreciativo + quebra da 4ª parede** (falar direto pra câmera) + ritmo acelerado; pattern interrupt constante (urso, facão) num plano-sequência. |
| **Squatty Potty — Unicorn** | Humor / educação de produto | **Choque visual + humor** para *educar sobre a dor*; metáfora absurda torna o problema memorável. Motion graphics + cortes rápidos. |
| **Old Spice — The Man Your Man Could Smell Like** | Surpresa / quebra de padrão | **Quebra de padrão constante** (cenário muda sem corte aparente); absurdo confiante. |
| **Always — Like a Girl** | Emocional / propósito | **Pergunta provocativa + propósito de marca**; emoção e identidade cultural geram compartilhamento. |
| **Dumb Ways to Die** | Engajamento / jingle | **Jingle chiclete + personagens fofos** contrastando com tema pesado; repetição que gruda. |

## Princípios destilados dos cases
1. **Prova > promessa**: demonstração inquestionável bate qualquer copy bonita.
2. **Emoção de alta excitação** (humor/admiração/raiva/inspiração) → compartilhamento.
3. **Quebra de padrão a cada 3–5s** mantém retenção.
4. **Falar com a câmera / direto com o espectador** cria intimidade e confiança.
5. **Uma ideia central forte** > muitas features.

## Adaptação para motion graphics (sem ator)
- "Demonstração extrema" → animação/diagrama que mostra o resultado de forma dramática
  (count-up, bar-grow, antes/depois com mask-reveal).
- "Quebra da 4ª parede" → texto em 2ª pessoa, grande, falando direto.
- "Plano-sequência dinâmico" → câmera virtual contínua (push-in/parallax) entre beats.

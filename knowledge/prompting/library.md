# Prompting — Biblioteca por categoria

Prompts-base por categoria, prontos para adaptar. Para `generated_still` (flux2-klein),
remova verbos de movimento e mantenha a descrição visual; o movimento vem do motion.

## Ação / adrenalina
```
Vídeo de 8s, comercial de ação, cortes secos, câmera dinâmica.
[0-2s] close em tênis batendo no asfalto molhado, tracking lateral, whip-pan, impacto sonoro
[2-4s] corredor em velocidade, handheld controlado, luz dura de fim de tarde, corte seco
[4-6s] push-in no rosto suado, expressão de foco, fundo em bokeh, batida da trilha
[6-8s] freeze no gesto de vitória, câmera fixa, grão de filme, halação quente
```

## Marketing / venda (produto)
```
Anúncio cinematográfico de produto, 15s, premium, ritmo médio-alto.
Produto sobre superfície escura, luz lateral dramática, reflexos sutis.
[0-3s] pull-back revelando o produto, halo de luz, partículas suaves
[3-8s] orbit lento mostrando detalhes, DOF raso, textura nítida
[8-12s] feature em texto grande surgindo, match-cut
[12-15s] logo + CTA, câmera fixa, LUT quente
```

## Viral / retenção (Reels)
```
Reel de 8s, vertical 9:16, jump cuts, legendas grandes, energia alta.
[0-2s] texto-gancho enorme entra com punch zoom, fundo de cor sólida vibrante
[2-5s] demonstração rápida, 3 cortes, pattern interrupt de cor a cada corte
[5-8s] CTA com seta animada apontando, whip-pan de saída
```

## Corporativo / B2B (autoridade)
```
Vídeo institucional, 16:9, elegante, sóbrio, ritmo calmo-confiante.
Ambiente clean, luz natural suave, paleta neutra com 1 accent.
[0-4s] push-in lento no apresentador/elemento, profundidade com bokeh
[4-10s] gráficos e números surgindo (count-up), transições por mask-reveal
[10-15s] frase-síntese em tipografia elegante, fade cinematográfico, CTA discreto
```

## Dica
Sempre incluir: **movimento de câmera específico**, **DOF/profundidade**, **iluminação**,
**acabamento (grão/LUT)**. Ver `prompting/best-practices.md` para checklist e erros.
